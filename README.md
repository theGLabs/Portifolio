# ☕ Café Guilado

Site de uma cafeteria fictícia criado como projeto de estudo de **HTML, CSS e JavaScript**.

## 📌 Sobre o projeto

O Café Guilado é um site de uma cafeteria com interface moderna que permite:

* Visualizar o cardápio
* Adicionar produtos ao carrinho
* Remover itens do carrinho
* Sistema de login simulado
* Layout com animações e design responsivo

## 🚀 Tecnologias usadas

* HTML5
* CSS3
* JavaScript
* Lottie Animations

## 📂 Estrutura do projeto

```
css/
js/
fotos/

index.html
cardapio.html
sobre.html
```

## 🎯 Objetivo

Praticar desenvolvimento front-end criando um site completo com interface, interatividade e organização de arquivos, simulando um projeto real de uma cafeteria.
